Module denverapi.cipher

This module provides cipher encrypting and decrypting of data. This contains many cipher methods. The Best of all is the vigenere cipher (but not better than AES Encryption provided by encryption module).

The Source code is taken from the book "Cracking Codes with Python". Although hacking methods for all the ciphers are not created, the hackers can still break the code by using different techniques, so it is recommended you should go for denverapi.encryption.
